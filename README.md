# Sales Data Analysis

This folder contains:

- `Sales_Analysis_Python_Code.py` - example analysis script using pandas.
- `Project_Report.docx` - project report with methodology, findings and recommendations.
- `sample_sales_dataset.csv` - dataset (located in `/Datasets`).

Run the Python script from the project root (so relative paths work):

```
python Projects/Sales-Data-Analysis/Sales_Analysis_Python_Code.py
```
